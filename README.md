# Install Project
``` 
mkdir scraping_app
cd scraping_app
python3 -m venv venvscraping_app
source venvscraping_app/bin/activate
```
## Clone git repository
```
git clone https://github.com/ahoysak/scraping_app.git
```
## Install requirements.txt
```
cd scraping_app
pip install -r requirements.txt
```
